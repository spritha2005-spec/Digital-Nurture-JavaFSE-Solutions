# E-commerce Platform Search Function

## Big O Notation
Big O notation is used to measure the efficiency of an algorithm as the input size grows.

## Search Operations

### Linear Search
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

### Binary Search
- Best Case: O(1)
- Average Case: O(log n)
- Worst Case: O(log n)

## Comparison

Linear Search checks each element one by one.

Binary Search repeatedly divides the sorted array into halves, making it much faster for large datasets.

## Conclusion

Binary Search is more suitable for an e-commerce platform because it provides faster search performance on sorted data.
