# Financial Forecasting

## Recursion

Recursion is a technique where a method calls itself to solve smaller instances of the same problem.

## Algorithm

The future value is calculated recursively using:

Future Value = Current Value × (1 + Growth Rate)

until the number of years becomes 0.

## Time Complexity

- Recursive Solution: O(n)
- Space Complexity: O(n)

where n is the number of years.

## Optimization

The recursive solution can be optimized using:
- Iteration (loop-based approach)
- Dynamic Programming (memoization)

These approaches reduce function call overhead and avoid excessive computation.
