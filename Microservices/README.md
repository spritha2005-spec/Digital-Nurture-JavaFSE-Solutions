# Microservices

## Description

This module demonstrates the development of independent Spring Boot microservices for a banking application.

## Microservices

- Account Microservice
- Loan Microservice

## Technologies Used

- Java 17
- Spring Boot
- Spring Web
- Maven

## Features

- Independent Spring Boot applications
- RESTful APIs
- Separate ports for each microservice
- Tested using Browser/Postman
