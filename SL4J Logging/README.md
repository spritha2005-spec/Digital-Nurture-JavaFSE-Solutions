# Exercise 1: Logging Error Messages and Warning Levels

## Objective

Demonstrate logging error messages and warning levels using SLF4J and Logback.

## Steps Performed

1. Added SLF4J and Logback dependencies to the Maven project.
2. Created a Java class named `LoggingExample`.
3. Configured a logger using `LoggerFactory`.
4. Logged an error message using `logger.error()`.
5. Logged a warning message using `logger.warn()`.

## Technologies Used

* Java
* Maven
* SLF4J
* Logback

## Output

ERROR LoggingExample - This is an error message

WARN LoggingExample - This is a warning message

BUILD SUCCESS
