# Exercise 1: Configuring a Basic Spring Application

## Objective

Configure a basic Spring application using XML-based configuration and dependency injection.

## Steps Performed

1. Created a Maven project named LibraryManagement.
2. Added Spring Core dependency in pom.xml.
3. Created applicationContext.xml for bean configuration.
4. Defined BookRepository and BookService beans.
5. Implemented dependency injection using setter injection.
6. Loaded the Spring Application Context in the main class.
7. Retrieved the BookService bean and executed the application.

## Technologies Used

* Java
* Spring Framework
* Maven
