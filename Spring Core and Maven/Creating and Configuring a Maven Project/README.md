# Exercise 4: Creating and Configuring a Maven Project

## Scenario

A new Maven project is created for the Library Management application and configured with Spring Framework dependencies.

## Objective

Set up a Maven project and configure the required Spring dependencies and Maven Compiler Plugin.

## Steps Performed

1. Created a Maven project named LibraryManagement.
2. Added Spring Context dependency.
3. Added Spring AOP dependency.
4. Added Spring WebMVC dependency.
5. Configured Maven Compiler Plugin.
6. Set Java source and target version to 1.8.
7. Verified the project build using Maven.

## Technologies Used

* Java
* Maven
* Spring Context
* Spring AOP
* Spring WebMVC

## Output

BUILD SUCCESS

## Result

The Maven project was successfully configured with the required Spring dependencies and Maven Compiler Plugin.
