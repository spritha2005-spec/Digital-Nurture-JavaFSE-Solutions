Exercise 4: Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown Methods in JUnit
Objective

Demonstrate the Arrange-Act-Assert (AAA) testing pattern and use setup and teardown methods.

Concepts Used
@Before
@After
assertEquals()
AAA Pattern
Arrange: Initialize input values.
Act: Call the method being tested.
Assert: Verify the result.
Test Fixtures
setUp() method initializes the Calculator object before each test.
tearDown() method cleans up resources after each test.
