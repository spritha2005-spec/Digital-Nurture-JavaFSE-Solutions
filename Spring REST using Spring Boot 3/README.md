# Spring REST using Spring Boot 3

## Description
This project demonstrates the basics of developing RESTful web services using Spring Boot 3.

## Features
- Hello World REST API
- Load country details from Spring XML configuration
- Fetch country details using REST endpoints
- Retrieve country information by country code (case-insensitive)

## Technologies Used
- Java 21
- Spring Boot 3
- Spring Web
- Maven
- Eclipse IDE

## Endpoints
- GET /hello
- GET /country
- GET /country/{code}

## Sample Response
```json
{
  "code": "IN",
  "name": "India"
}
```
