# Hello World RESTful Web Service

## Objective
Create a simple RESTful web service using Spring Boot that returns the text **"Hello World!!"**.

## Endpoint

- **Method:** GET
- **URL:** http://localhost:8083/hello

## Response

```
Hello World!!
```

## Features
- REST Controller using `@RestController`
- GET endpoint using `@GetMapping`
- SLF4J logging with start and end messages
- Tested using Chrome Browser and Postman

## Technologies Used
- Java 17
- Spring Boot
- Spring Web
- Maven
