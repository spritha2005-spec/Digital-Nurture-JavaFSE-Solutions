# Create a Spring Web Project using Maven

This exercise demonstrates how to create a Spring Boot web application using Maven and Spring Initializr.

## Objectives
- Create a Spring Boot project using Spring Initializr
- Configure Maven dependencies
- Add Spring Boot DevTools and Spring Web
- Verify application startup using logging
- Run the application with the embedded Tomcat server

## Technologies Used
- Java 17
- Spring Boot
- Spring Web
- Maven
