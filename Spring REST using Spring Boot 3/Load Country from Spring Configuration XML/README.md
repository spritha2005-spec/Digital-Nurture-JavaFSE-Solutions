# Spring Core – Load Country from Spring Configuration XML

This exercise demonstrates loading a Spring bean from an XML configuration file using the Spring IoC container.

## Objectives
- Configure a bean in `country.xml`
- Create a `Country` class with logging
- Load the bean using `ClassPathXmlApplicationContext`
- Retrieve the bean using `ApplicationContext`
- Display the country details using Spring Core

## Technologies Used
- Java 17
- Spring Boot
- Spring Core
- Maven
